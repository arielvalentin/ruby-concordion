set

Here is an example of how to set a variable and use it in a subsequent tag.

If my username is <span concordion:set="#username">Chuck Norris</span>
Then the system should greet me with <span concordion:assertEquals="greetingFor(#username)">Hello Chuck Norris, you nancy-boy.</span>

The relevent fragments of the tag are:

  • set - tells concordion to bind the variable (whose name begins with #)
  • #username - the concordion variable name that will be bound to the tag contents
  • Chuck Norris - the value of the variable
  • greetingFor(#username) - invocation of a fixture method with the variable as an argument.
